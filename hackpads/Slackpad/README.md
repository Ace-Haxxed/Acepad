# Slackpad

Simple Hackpad based on RP2040 XIAO.

- 1x Seeed Studio XIAO RP2040 or BLE
- 1x 0.91" OLED Screen
- 7x Blank DSA keycaps (would it be possible to get spares as well?)
- 2x Screws & heatsets
