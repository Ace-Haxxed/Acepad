MacroPad V.1

This is a simple macropad with 4 switches and 2 rotary encoders. Each having the following functions:

------Switches------

1.) Ctrl+A
2.) Ctrl+C
3.) Ctrl+V
4.) LED Color and Pattern Control

------Rotary Encoders------

5.) LED Brigthness
6.) Voulme Control 

BOM:
4x Cherry MX Brown switches
2x EC11 Encoder
3x PCB (Black & Gold)
4x Through-hole 1N4148 Diodes
6x WS2812B LEDs
